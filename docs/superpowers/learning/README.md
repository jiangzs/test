# Superpowers 技能学习课程

## 课程目标
完成本课程后，你将能够：
1. 理解 Superpowers 技能系统的设计哲学
2. 在正确的时间调用正确的技能
3. 遵循每个技能的最佳实践
4. 使用技能系统完成复杂的开发任务

## 课程目录

| 章节 | 主题 | 预计时间 |
|------|------|----------|
| 01 | 核心概念与规则 | 15 分钟 |
| 02 | 头脑风暴 (brainstorming) | 20 分钟 |
| 03 | 编写计划 (writing-plans) | 20 分钟 |
| 04 | 执行计划 (executing-plans) | 15 分钟 |
| 05 | 测试驱动开发 (test-driven-development) | 30 分钟 |
| 06 | 系统调试 (systematic-debugging) | 20 分钟 |
| 07 | 代码审查 (requesting/receiving-code-review) | 20 分钟 |
| 08 | 子代理协作 (dispatching-parallel-agents) | 15 分钟 |
| 09 | 验证完成 (verification-before-completion) | 10 分钟 |
| 10 | Git Worktrees | 15 分钟 |
| 11 | 综合实践项目 | 60 分钟 |

## 学习建议
- 按顺序学习，每个技能建立在前一个之上
- 完成每个章节的练习后再继续
- 遇到实际问题时，回顾相关技能章节
