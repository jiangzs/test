## 项目概述

为斐波那契项目开发一个命令行工具 (CLI)，主要用于教学演示场景。重点在于清晰的输出、友好的提示和交互式学习体验。

| 组件 | 技术 | 理由 |

## 命令设计

### 主命令：fibonacci

```bash
# 基础用法
fibonacci 10              # 计算第 10 个斐波那契数
fibonacci sequence 10     # 显示前 10 项序列
fibonacci interactive     # 进入交互模式

# 帮助信息
fibonacci --help          # 显示详细帮助
fibonacci --guide         # 启动新手引导
```

## 输入验证

**规则:**
- 索引必须是非负整数 (n >= 0)
- 序列计数必须是正整数 (count >= 1)
- 最大值限制：n <= 1000（防止计算过慢）

**错误提示:**
```bash
$ fibonacci -5
❌ 错误：索引 n 必须是非负整数
💡 提示：斐波那契数列从 0 开始，如 fibonacci 0, fibonacci 1, ...

$ fibonacci sequence 0
❌ 错误：序列计数必须是正整数
💡 提示：请指定至少 1 项，如 fibonacci sequence 1
```
